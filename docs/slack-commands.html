슬랙에서 사용 가능한 모든 명령어와 기능 안내 | 최종 업데이트: 2026-03-27

가끔 사용
관리용

1. 주문 관리

명령어 설명 결과 타입
/order 최근 주문 목록 (5건씩 페이지네이션) 주문 리스트 + [상세] [이전/다음] 목록
/order [이름] 고객명으로 주문 검색 예: /order 강동현 검색
/order [상태] 상태별 필터 (대기/진행중/완료/취소) 예: /order 진행중 검색
/order 추가 주문 등록 모달 주문자/상품/수량/배송지 입력 → 등록 모달

2. 문자 (SMS)

명령어 설명 결과 타입
/sms 문자 발송 모달 받는 사람 검색 → 대화 내역 → 메시지 입력 → 발송 모달
/sms [번호] [메시지] 문자 바로 발송 예: /sms 010-1234-5678 내일 배송 예정입니다 바로발송
/sms [이름] [메시지] 이름으로 발송 예: /sms 강동현 내일 배송 예정입니다 바로발송

3. 오퍼레이션 (작업 현황)

명령어 설명 결과 타입
/operation 미배정/접수/제작/배송중 단계별 주문 + [상세] 버튼 칸반 보드 (전체 현황) 칸반
/operation [단계명] 특정 단계 상세 예: /operation 접수 → 접수 주문 + 버튼들 상세

4. 연락처

명령어 설명 결과 타입
/contact 전체 연락처 목록 (타입별 그룹핑) 이름, 연락처, 주소, 메모 표시 목록
/contact [이름 또는 번호] 연락처 검색 예: /contact 강동현 또는 010-9533 검색
/contact 추가 연락처 등록 모달 이름, 분류, 전화번호, 주소, 메모 입력 모달
/contact 수정 [이름] 연락처 수정 [수정] 버튼 → 수정 모달 모달
/contact 삭제 [이름] 연락처 삭제 정확한 이름 필요, 여러 명이면 안내 텍스트

5. 재고

명령어 설명 결과 타입
/stock 전체 재고 현황 품목별 수량, 부족 여부 목록
/stock 부족 부족 재고만 표시 최소 기준 이하 항목만 목록
/stock [SKU 또는 이름] 특정 품목 상세 예: /stock CANVAS-300 상세
/stock-in 입고 처리 모달 품목, 수량, 사유 입력 모달
/stock-out 출고 처리 모달 품목, 수량, 사유 입력 모달

6. 대시보드 / 프로필

명령어 설명 결과 타입
/dashboard 운영 현황 대시보드 재고 현황 + 진행중 주문 요약 요약
/profile 프로필 목록 이름, 설명, 상품, 필수 연락처 + [수정] 목록
/profile [이름] 프로필 검색 예: /profile 표준 검색
/profile 수정 [이름] 프로필 수정 (이름/설명만) [수정] 버튼 → 수정 모달 모달

7. 연락처 타입 (관리용)

명령어 설명 결과 타입
/contact-type 타입 목록 고객, 화물/배차 등 + 기본 여부 목록
/contact-type 추가 타입 추가 모달 이름, slug 입력 모달
/contact-type 추가 [이름] [slug] 타입 인라인 추가 예: /contact-type 추가 원단업체 fabric 텍스트
/contact-type 삭제 [이름] 타입 삭제 기본 타입은 삭제 불가 텍스트

8. 주문 상세 모달 버튼

주문 목록에서 [상세] 클릭 시 열리는 모달 안의 버튼들

버튼 설명 조건
접수 시작 미배정 주문을 접수 단계로 이동 미배정 상태일 때만 표시
연락처 배정 미배정 연락처 타입별 검색 → 배정 미배정 연락처가 있을 때
체크리스트 작성 현재 단계 체크리스트 체크/입력 미완료 체크리스트가 있을 때
보내기 템플릿별 SMS 발송 (대화 내역 표시) 템플릿이 있을 때, 템플릿별 개별 버튼
삭제 주문 삭제 (확인 다이얼로그) 항상 표시

9. 오퍼레이션 단계 상세 버튼

/operation에서 [단계 상세] 클릭 시 나오는 주문별 버튼들

버튼 설명
주문 상세 주문 상세 모달 열기
체크리스트 체크리스트 작성 모달
다음 단계로 다음 단계로 이동 (마감일 선택 가능, 체크리스트 미완료 시 경고)
완료하기 마지막 단계에서 주문 완료 처리
접수 시작 미배정 상세에서 접수 단계로 이동

10. 자동 알림 (웹훅)

CS Tool에서 변경 시 자동으로 슬랙 채널에 알림이 옵니다

채널 이벤트 알림 내용
#알림-주문 새 주문 등록 주문자 / 상품 / 단계 / 주소 / 납기
#알림-주문 주문 삭제 주문자 / 상품
#알림-오퍼레이션 단계 이동 접수 → 제작 등
#알림-오퍼레이션 마감일 변경 이전 마감 → 새 마감 (구캘 연동)
#알림-오퍼레이션 상태 변경 진행중 → 완료 등
#알림-재고 입출고 품목 / 수량 / 사유
#알림-재고 재고 부족 품목 / 현재수량 / 기준수량
#오퍼레이션-채팅 SMS 발신 받는 사람 / 내용 / 발신자
#오퍼레이션-채팅 SMS 수신 보낸 사람 / 내용 + [답장] [연락처 등록] 버튼
